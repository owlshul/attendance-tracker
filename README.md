# Attendance Tracker App

A modern, dual-mode attendance tracking application built with React.

## Features
- Simple Mode: Quick manual entry
- Advanced Mode: Detailed subject-wise tracking
- Interactive forecast charts
- Persistent storage
- Beautiful, responsive UI

## Deploy to Netlify

### Method 1: Netlify Drop (Easiest)
1. Go to https://app.netlify.com/drop
2. Drag and drop the entire `attendance-app` folder
3. Your site will be live in seconds!

### Method 2: GitHub + Netlify (Recommended for updates)
1. Create a GitHub repository
2. Push this code to GitHub
3. Go to https://app.netlify.com
4. Click "Import from Git"
5. Connect your GitHub repo
6. Netlify will auto-deploy on every push

### Method 3: Netlify CLI
```bash
npm install netlify-cli -g
netlify deploy --prod
```

## Local Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

The built files will be in the `dist` folder.
